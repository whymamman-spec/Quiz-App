# Quiz App (JavaScript Project)

## 📌 Project Overview

This is a simple interactive Quiz Application built using HTML, CSS, and JavaScript. The app presents multiple-choice questions to the user, tracks their answers, provides instant feedback, and displays the final score at the end of the quiz.

The project is designed to demonstrate core JavaScript fundamentals including arrays, objects, functions, loops, and conditional logic in a real-world interactive application.

---

## 🚀 Features

* Displays questions one at a time
* Multiple-choice answer selection using buttons
* Instant feedback after each answer (Correct / Wrong)
* Score tracking throughout the quiz
* Final results screen at the end of the game
* Simple and responsive user interface

---

## 🛠️ Tools & Technologies Used

* HTML5 – Page structure
* CSS3 – Styling and layout design
* JavaScript (Vanilla JS) – Game logic and interactivity
* VS Code – Development environment
* Browser Developer Tools – Debugging and testing

---

## 🧠 Key Concepts Demonstrated

### 1. Arrays & Objects

* Used to store quiz questions, options, and correct answers
* Structured data management for scalability

### 2. Functions

* Modular functions used to:

  * Display questions
  * Check answers
  * Update score
  * End the quiz

### 3. Loops

* Used to dynamically generate answer buttons
* Eliminates repetition in UI creation

### 4. Conditional Logic

* Determines correct vs incorrect answers
* Controls game flow (progression and end state)

### 5. DOM Manipulation

* Dynamically updates:

  * Questions
  * Score
  * Feedback messages

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Launch `index.html` in any modern web browser
4. Start answering the quiz questions

No additional installations are required.
